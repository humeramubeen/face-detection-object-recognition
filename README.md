# Face Detection & Object Recognition

A modern AI-powered web application that performs real-time face detection and object recognition using JavaScript and TensorFlow.js.

## Features

- Real-time webcam detection
- Face Detection
- Object Recognition
- Responsive UI
- Fast Performance

## Technologies

- HTML
- CSS
- JavaScript
- TensorFlow.js
- face-api.js

## Installation

Clone the repository

```bash
git clone https://github.com/humeramubeen/face-detection-object-recognition.git
```

Open

```
index.html
```

or use Live Server.
